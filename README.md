💡 Описание проекта

Этот проект представляет собой консольное приложение для управления личными финансами.
Пользователь может создавать учетную запись, добавлять доходы, расходы, категории, устанавливать бюджеты, просматривать баланс и сохранять данные в файл.

🧩 Основные возможности

- 🔐 Авторизация и регистрация пользователя
- 💰 Добавление доходов и расходов
- 🏷️ Создание, удаление и переименование категорий
- 📊 Установка лимита бюджета на категории
- 💹 Просмотр статистики по кошельку и по категориям
- 🔄 Переводы между пользователями
- 🗂️ Сохранение данных в файл

⚙️ Технологии

- Java 17+
- Работа с потоками (Stream API)
- Обработка исключений
- Простая файловая сериализация данных

📄 Пример работы

При запуске программа предложит ввести логин и пароль.
Если пользователь не найден — создается новый аккаунт.
После входа в систему откроется меню с выбором действий по управлению финансами.
